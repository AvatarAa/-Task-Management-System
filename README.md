
# Task Management System

## Overview
A PHP-based task management system designed to allow users to create, edit, complete, and manage tasks through a structured dashboard interface.

## Features
- User registration and login
- Add tasks
- Edit existing tasks
- Mark tasks as completed
- Delete tasks
- Dashboard for task tracking
- Database integration

## Technologies Used

- PHP
- SQL
- HTML/CSS
- Git/GitHub

## File Structure

```text
index.php          -> Entry point
register.php       -> User registration
dashboard.php      -> User dashboard
add_task.php       -> Create tasks
edit_task.php      -> Update tasks
complete_task.php  -> Mark task complete
delete_task.php    -> Remove task
db.php             -> Database connection
```
